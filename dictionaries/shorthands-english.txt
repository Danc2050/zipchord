This is a sample starting English-language shorthand dictionary for ZipChord. It contains the most frequent English words.
See https://github.com/psoukie/zipchord for details.

This dictionary is based on the list of 5,000 most frequent English words at https://en.wiktionary.org/wiki/Wiktionary:Frequency_lists/A_Frequency_Dictionary_of_Contemporary_American_English) which has 4374 entries after removing duplicates.

The format of this file is as follows: Sequences of typed keys that trigger the shorthand recognition are separated from the expanded full words by Tab. All other lines are ignored by ZipChord. You can change existing shorthands and add your own.
Note: If you add a shorthand for an existing entry with "(no shorthand)", remove the note in the parantheses, otherwise it will be included in the expanded text.

The logic behind the shortcuts in this dictionary adheres to consistent principles:
1. If the word is three characters or less, no shorthand is predefined
2. Doubled consonants (two consonants immediately following each other) are replaced with only one consonant
3. If the word starts with a vowel, the vowel is retained in the shorthand
4. All remaining vowels are removed
5. If the resulting shorthand is already in use or is a word in its own right, no shorthand is predefined

For example, "actually" is defined as "actl", and "career" as "crr" (and slightly trickier: "attitude" is "attd")
